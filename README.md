# Monkeycrash
A gambling crash game made for a school project with pygame

# Download & install
Dependencies: Python, Pygame
Pygame can be download with pip:

`pip install pygame`
or executing "**dependecies.py**"

**[Download the Project][1]**

## Run
Just have to run "**scene.py**"
